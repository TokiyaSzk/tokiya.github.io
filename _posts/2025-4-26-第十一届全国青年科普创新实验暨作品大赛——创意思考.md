## 一、自然场景
*重点：生态变化 生物活动 生态恢复 高效检测*  

该场景主要涉及到自然环境的检测和恢复方案，对额外的传感器硬件需求较高
在**数据可视化**，检测**数据上云**等多设备互联统一控制检测可能会有创新

但现行类似气象台和环境治理等部门已有成熟的土壤、大气环境等专业的检测系统。


|               项目                |         技术         | 硬件支持 |
| :-------------------------------: | :------------------: | :------: |
| [区域流浪猫检测](#区域流浪猫检测) | 视觉识别、多设备互联 |  摄像头  |
  
### 区域流浪猫检测

**目的**  
 
小区、社会街道等等地区流浪猫的数量日益增多。现社会有许许多多的公益组织在救助流浪猫
为流浪猫进行驱虫，绝育等。但是儿童和老人的身体健康问题，流浪猫传染病和伤人等等问题依旧存在。
通过设计一套定点投放救助猫粮，利用视觉识别技术为每一个流浪猫进行特定识别入库。
及时告警怀孕、猫瘟等易引发健康安全事件的流浪猫。检测每个区域的新增流浪猫的数量，将
数据可视化上云，为公益组织提供数据等支持。
***怀孕期流浪猫易发生伤人事件***

**功能**  

|        功能        |                 详情                 |
| :----------------: | :----------------------------------: |
|   流浪猫个体识别   |          统计区域流浪猫数量          |
|   流浪猫标签功能   |  设置已绝育、已驱虫、怀孕等特征标签  |
| 新怀孕、传染病告警 |       及时告警怀孕、传染病猫只       |
|     数据可视化     | 可视化已绝育数量、流浪猫新增减少数量 |

### 文化宣传国际化

**目的**  

利好国内文旅企业，在宣传国内文化之时，利用RAG和生成式AI，将国内宣传内容利用国外流行文化倾向
自动生成面向国外的宣传方案、图文、视频等内容。减少为外国游客特定制作宣传内容的工作。

**功能**

|       功能       |                        详情                        |
| :--------------: | :------------------------------------------------: |
| 国外流行文化收集 |           收集国外流行语言、社会流行内容           |
| 目标游客国家选择 |          根据针对不同的国家生成不同的内容          |
|     文案生成     |         利用RAG，生成目标国家本土化的文案          |
|      宣传图      |     根据目标国家的文化倾向制作不同风格的宣传图     |
|     视频剪辑     | 根据目标国家的文化倾向，利用源视频制作不同风格视频 |

### 传统文化流行化宣传  

该内容具有一定局限性。使用AI设计的传统文化宣传真的有确实效果吗？真的能将传统文化转变为
吸引人们的宣传内容吗？

### 传统文化AR&VR体验

**目的**  

社会上对传统文化有一定的兴趣，但是苦于传统媒体的宣传力以及对于年轻人的吸引力较低
许多人会选择放弃了解传统文化，转而投向流行文化。利用VR&AR设备将其变成看的见摸得着的内容，
吸引更多的群众来了解、学习传统文化

**功能**

|     功能     |                        详情                         |
| :----------: | :-------------------------------------------------: |
| 传统文化介绍 | 利用VR&AR，身临其境参与到**传统文化节日**的体验之中 |
| 非遗文化体验 |     利用VR&AR，亲身的去学习**非遗文化**制作技艺     |

### 全过程个人学习助理

**目的**  

在教育学习的过程当中，最火热的术语就是差异化教学。利用RAG、长期记忆技术，并且动态
处理RAG、长期记忆中的学生当前学习的情况。使用prompt工程确保多模态模型正确合适的
给予建议，并且利用格式化的数据，自动客制化不同学科，不同年龄段，不同学情的学生学习计划

**主要功能**  
根据个人身体状况，心理倾向，推荐不同的体育运动。生成不同体育运动的运动计划，并支持客制化，并使用交互界面帮助用户


### 今日运动

**主要功能**  
根据今日的心情，天气，已有器材，周边设施条件，经济状况，身体状况与计划运动时间等等。推荐今日体育运动项目与细节计划
